# AI Practitioner题库 - 手机安装说明

APP网址：https://ercsion.github.io/aws-ai-practitioner-app/

这是一个网页APP（PWA），安装后可以像普通APP一样在主屏幕上打开，支持全屏运行和离线使用（联网打开一次后，没网也能背题/测试）。

---

## iPhone（Safari浏览器）

1. 用 **Safari** 打开上面的网址（必须是Safari，其他浏览器没有这个功能）
2. 点击底部工具栏中间的分享按钮（方框带箭头图标）
3. 下拉菜单，找到并点击 **"添加到主屏幕"**
4. 确认添加，主屏幕上会出现"AI Practitioner题库"图标

---

## 安卓手机（Chrome浏览器）

1. 用Chrome打开上面的网址
2. Chrome通常会自动在页面底部弹出"安装应用"提示条，直接点击安装
3. 如果没有自动弹出，点击右上角菜单（三个点）→ 找到 **"安装应用"** 或 **"添加到主屏幕"**，点击确认

### 如果点击后没有反应（常见于小米/红米 MIUI 系统）

MIUI系统可能默认关闭了Chrome创建桌面快捷方式的权限，需要手动开启：

1. 打开手机 **设置 → 应用设置 → 权限管理**
2. 找到 **Chrome**，点进去
3. 找到 **"创建桌面快捷方式"** 权限，开启它
4. 回到Chrome重新打开网址，再次尝试安装

### 保底方案（如果Chrome始终不行）

用手机**自带的浏览器**（不是Chrome，是系统自带的那个）打开同一个网址，点击浏览器菜单里的"添加到桌面"，通常兼容性更好，几乎不会失败。

---

## 安装后如何使用

- 点击主屏幕图标打开，全屏显示，没有浏览器的地址栏和工具栏
- 首次打开需要联网加载一次，之后可以离线使用
- 如果之后APP有更新，重新打开一次即可自动获取最新版本（如遇内容没更新，可尝试完全关闭APP后重新打开）

---

## 常见问题

**Q: 安装后找不到图标？**
去手机的"应用列表"（不是主屏幕，是完整应用抽屉）里搜索"AI Practitioner"确认是否已安装。如果确实没有，说明安装没成功，请重新走一遍上面的步骤。

**Q: 页面打不开或加载很慢？**
GitHub Pages在国内访问偶尔不稳定，可以稍等重试，或检查网络连接。
